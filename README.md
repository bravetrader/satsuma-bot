---

### 1. Clone this git

```
git clone https://github.com/bravetrader/satsuma-bot.git
cd satuma-bot
```
### 2. Create venv

 ```
python3 -m venv venv
```
### 3. Install dependcies

```
  pip3 install -r requirements.txt
```
### 4. Run the bot

```
python3 bot.py
```
### 5. Enjoy!
